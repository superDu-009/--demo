## 风险与应对

| 风险 | 影响 | 应对 |
|------|------|------|
| Seedance API 不通/限流 | 高 | 提前验证 API Key 和调用配额，准备 Mock 方案 |
| TOS 上传失败 | 高 | 本地文件系统 fallback |
| 流程引擎复杂度高 | 中 | 先做最简版（线性执行），断点续跑可延后 |
| 前端页面开发慢 | 中 | 先保证功能，UI 美观度可延后 |
| AI 生图质量差 | 中 | 准备预设 Prompt 模板，保证基本效果 |
