## 1. 背景与目标
### 1.1 业务背景
个人创作者（蓝烟老师及朋友）需要将网络小说快速转化为 AI 漫剧（视频形式的短剧）。核心目标：**保证视频质量的前提下，跑通「小说→手动拆分结构→资产提取→生成分镜→生图生视频」全手动可控流程**，所有操作由用户主动触发，无自动执行逻辑，保障生成内容完全可控。
### 1.2 排期说明
| 版本 | 内容 | 排期 |
|------|------|------|
| V1.2本期 | 核心流程跑通：上传剧本→拆分集→拆分镜→资产提取→生图生视频 | 本期上线 |
| V2.0二期 | 成本统计、风格预设后台配置、大模型prompt后台配置 | 二期开发 |
| V3.0三期 | 自动流程引擎、剪辑功能 | 三期开发 |
### 1.3 全局硬约束（绝对不能违反）
✅ 所有文件（小说/参考图/音频/生成内容）全部前端预签名直传火山TOS，后端不接收文件流，不存本地，仅存TOS路径，文件永久保留不删除
✅ 生图/生视频必须强制传入绑定资产的主参考图参数，保障人物/场景100%一致性
✅ 所有可选选项全部做下拉选择，尽量减少用户手动输入
✅ 所有可配置参数、prompt、生成内容全部落库，绝对不硬编码在代码中，100%灵活可配置
✅ AI返回模板允许放静态资源或配置文件维护，但绝对不能写死在业务代码中
✅ 所有数据删除均为软删，不物理删除数据
✅ 取消原PRD的项目状态、分场层级、自动流程引擎、成本统计四个模块/功能

## 2. 用户与核心流程
### 2.1 核心用户
仅面向内部创作者使用，无对外C端用户
### 2.2 完整操作流程
1. 登录→新建项目→上传小说（txt/word/md格式）→填写项目参数
2. 进入剧本预览页→点击「剧本分析」AI拆分集→手动调整分集
3. 进入分镜工作台→点击「分镜拆分」AI拆分单集分镜→手动调整分镜内容/绑定资产
4. 进入资产库→点击「提取资产」AI提取分集资产→手动调整资产关系/生成资产图片
5. 回到分镜工作台→手动触发生成分镜图片/视频→审核内容→单个下载需要的素材

## 3. 功能需求
### 3.1 模块1：登录/个人中心
#### 基础功能
- 仅支持账号密码登录，无注册/忘记密码功能，账号后台手动创建
- Sa-Token鉴权，所有业务接口必须带有效token，密码BCrypt加密存储
- 连续输错5次密码锁定账号10分钟，token有效期24小时
#### 优化功能
- 登录成功弹窗文案：「欢迎你, {用户名}~」
- token过期不跳转登录页，弹出登录弹窗，登录完成后留在当前页面，不丢失上下文
- 右上角头像点击弹出个人中心：支持上传头像、修改用户名、修改密码
### 3.2 模块2：项目管理
#### 项目列表页
- 卡片布局，显示项目名称、缩略图、创建时间，支持搜索、新建、编辑、删除（软删）
- 项目缩略图规则：选预设风格自动匹配对应风格默认图，自定义风格用通用默认图
#### 新建/编辑项目页
- 必填字段：项目名称、小说上传（支持txt/word/md，最大100M，仅提取纯文本，忽略图片）
- 可选参数（下拉选择，生成时可修改）：
  - 比例：16:8 / 16:9 / 4:3 / 9:16（默认值，生成时可修改）
  - 清晰度：720P / 1080P / 2K / 4K（默认值，生成时可修改）
  - 风格：2D次元风 / 日漫风 / 国漫风 / 古风 / 现代写实 / 自定义（项目创建后固定不可修改）
- 自定义风格：填写风格描述，自动作为前缀加到所有生图/生视频提示词中，保障全局风格统一
- 小说仅新建时可上传，编辑时不可修改
#### 项目详情页
固定3个Tab，顺序不可调整：【剧本预览】→【分镜工作台】→【资产库】
### 3.3 模块3：剧本预览（Tab第1位）
#### 基础功能
- 自动解析上传的小说内容，分页显示：每页500字，点击页面左右下角箭头切换上下页，模仿小说阅读交互
- 原剧本内容永久保留，所有拆分操作不修改原剧本
#### 核心操作
- 右上角「剧本分析」按钮：点击弹窗选择拆分规则（prompt后续补充，后端留可配置空位），调用大模型拆分原剧为多个分集
- 解析状态：待解析→解析中→解析成功→解析失败，失败显示具体原因，支持手动点击重试，不自动重试
- 拆分完成后自动跳转到分镜工作台，左侧生成对应分集列表
### 3.4 模块4：分镜工作台（Tab第2位，核心操作区）
#### 布局结构
- 左侧：二级层级树「项目→分集」，每个分集显示名称+概述，点击展开icon可查看分集完整对应剧本内容+该分集关联的所有资产列表
- 右侧：分镜卡片列表，超过100条自动开启虚拟滚动，支持按状态筛选、按序号排序
- 每个分集右上角操作按钮：「分镜拆分/重新拆分」（未拆分显示拆分，已拆分显示重新拆分，拆分时弹二次确认：「重新拆分将覆盖现有分镜，是否确认？」）
- 拆分规则：默认按每个分镜不超过15s拆分，支持下拉选择10s/12s/15s，大模型prompt可配置
#### 分镜核心规则
- 层级：「项目→分集→分镜」三级，完全取消分场
- 分镜字段：序号、中文提示词、英文提示词、时长、景别、运镜、台词、生成图片URL、生成视频URL、尾帧URL、关联资产列表、承接上一分镜开关、草稿内容
- 大模型拆分时自动返回景别、运镜、台词字段，全部入库支持手动编辑
- 分镜无统一状态，三个独立子状态：提示词生成状态（未生成/生成中/生成成功/生成失败）、图片生成状态（未生成/生成中/生成成功/生成失败）、视频生成状态（未生成/生成中/生成成功/生成失败）
#### 丝滑衔接核心逻辑（保障视频不跳戏）
- 每个分镜新增「承接上一分镜」开关，默认开启，转场到新场景时手动关闭
- 开关开启时：
  1. 分镜提示词自动拼接上一分镜结尾内容，保障剧情连续
  2. 生成视频时自动传入上一分镜的尾帧URL作为Seedance API的`first_frame`参数，保证画面衔接
- 开关关闭时：独立生成，不承接上一分镜内容
#### 操作功能
- 支持手动新增、编辑、删除分镜，拖拽调整分镜顺序
- 编辑分镜弹窗：支持修改提示词、景别、运镜、台词、绑定/解绑资产、开关承接上一分镜、保存草稿
- 顶部批量操作按钮：批量生成提示词、批量生成图片、批量生成视频、批量通过、批量打回
- 所有生成类按钮逻辑统一：未生成→「生成」，生成成功→「重新生成」，生成失败→「重试」
- 生成参数修改：生图/生视频时可修改比例、清晰度，修改视频参数与全局默认不一致时弹二次确认：「当前参数与全局配置不一致是否确定生成」
- 生成失败显示具体错误原因，不自动重试，用户修改后手动点击重试
- 单个分镜支持：图片预览下载、视频预览下载
### 3.5 模块5：资产库（Tab第3位，一致性核心）
#### 布局结构
- 顶部固定4个Tab，顺序：角色→场景→道具→声音
- 卡片网格布局，显示资产缩略图、名称、描述、操作按钮，支持搜索、批量操作
- 父资产卡片缩略图旁展示所有关联子资产的小缩略图，子资产独立在列表中存在
#### 核心操作
- 顶部「提取资产」按钮：点击弹窗选择要解析的分集，调用大模型提取该分集的4类资产，仅生成资产描述，不自动生成图片，每个资产显示默认缩略图+名称+简短概述
- 提取时自动检测重复资产：名称/描述相似度超过80%弹出提示，用户可选择合并/删除/保留，避免资产幻觉
- 资产关系：多对多关系，支持一个子资产归属于多个父资产，新建资产时可选择是否为子资产，选择对应父资产，关系支持手动编辑
- 资产卡片操作按钮：
  - 生成：点击按当前提示词生图，默认图显示「图片生成中」，成功后替换为生成图
  - 编辑：弹窗支持修改名称/描述、选择参考图（父资产图/本地上传，最多3张）、调整父子归属关系
  - 删除：被分镜绑定的资产禁止删除，弹出提示
#### 约束规则
- 每个资产至少上传1张参考图才可标记为已确认
- 仅已确认的资产可被分镜绑定
- 被分镜绑定的资产禁止删除，但允许继续修改名称、描述、参考图和父子归属关系；修改后后续生成默认使用最新资产信息

## 6. 技术方案
### 6.1 技术栈（无调整）
| 层 | 技术 |
|----|------|
| 后端 | Java 17+, Spring Boot 3.2, MyBatis Plus 3.5.5+, Sa-Token 1.37, Redis |
| 前端 | Vue 3 + TS, Vite, Element Plus, SortableJS, vue-virtual-scroller |
| 数据库 | MySQL 8.0 |
| 存储 | 火山 TOS（AWS S3 兼容 SDK） |
| AI调用 | Seedance 2.0 API（HTTP）, 文本大模型（GPT-4o/Claude） |
### 6.2 大模型接口规则
所有涉及文本处理的大模型调用接口，后端全部留可配置的prompt空位，prompt修改无需改代码：
1. 剧本拆分集接口prompt
2. 分集拆分镜接口prompt
3. 资产提取接口prompt
4. 分镜提示词生成接口prompt
5. AI返回模板可放静态资源维护，但不得写死在业务代码中
### 6.3 Seedance 调用规则
生成视频时，若「承接上一分镜」开关开启，自动传入上一分镜的`last_frame_url`作为`first_frame`参数，同时传入所有绑定资产的主参考图作为`reference_image`参数

## 7. 验收标准
1. 登录过期弹出登录窗，登录后不跳转不丢编辑内容
2. 上传word/md小说，忽略图片，纯文本解析正确
3. AI拆分集/分镜失败显示具体错误，支持手动重试，同时支持完全手动新增/编辑/删除分集/分镜
4. 提取资产时自动检测重复，弹出提示让用户选择处理方式
5. 分镜开启承接开关时，生成视频自动传入上一分镜尾帧，画面衔接无跳戏
6. 生成分镜视频成功后，自动保存尾帧到TOS
7. 修改视频参数与全局不一致时弹出二次确认
8. 所有生成按钮状态符合逻辑：未生成→生成，成功→重新生成，失败→重试
9. 被分镜绑定的资产无法删除
10. 所有参数、prompt、生成内容全部落库，无硬编码；AI返回模板允许放静态资源维护，但不能写死在业务代码中
